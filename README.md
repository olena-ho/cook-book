# Cook Book

Cook Book is a full-stack pet project for managing and browsing recipes.

## Tech stack

### Backend

- .NET
- ASP.NET Core Web API
- Clean Architecture
- CQRS with MediatR
- Entity Framework Core
- PostgreSQL

### Frontend

- React
- TypeScript
- Vite
- SCSS Modules

### DevOps

- Docker
- Docker Compose
- Kubernetes later
